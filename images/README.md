# Image Assets for GitHub.io Portfolio

This directory contains images for the portfolio website.

## Required Images

1. **profile.jpg** - Your professional profile photo
   - Recommended size: 400x400px
   - Format: JPG or PNG
   - Should be professional and high-quality

## Optional Images

2. **og-image.jpg** - Social media preview image (Open Graph)
   - Recommended size: 1200x630px
   - Used when sharing on social media

3. **favicon.ico** - Website favicon
   - Size: 32x32px or 16x16px
   - Format: ICO or PNG

## Adding Your Images

1. Add your profile photo as `profile.jpg` or `profile.png`
2. Update the HTML if needed to reference your images
3. Ensure images are optimized for web (compressed, appropriate size)

## Image Optimization Tips

- Use tools like TinyPNG or ImageOptim to compress images
- Use modern formats like WebP for better performance (with fallbacks)
- Consider using lazy loading for better performance
- Use appropriate image sizes for different screen resolutions
